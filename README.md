# PixelCrafters - Agencia Digital Creativa

Proyecto final de Diseño Web - 100% funcional y responsive

## Características
- Video de YouTube full-screen como fondo del hero
- Tema claro/oscuro con botón
- Menú hamburguesa en móviles
- Animaciones al hacer scroll
- Portafolio con efectos hover
- Formulario de contacto

## Tecnologías
- HTML5 + CSS3 + JavaScript puro
- Google Fonts + multimedia
